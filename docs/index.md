# Documentation index

Codex Pets Desktop is a standalone Electron and Angular app.

It follows Codex Pets behavior and keeps compatibility with Codex Pets assets and pet installation flow.

## Documents

– `architecture.md` – project architecture, app layers, IPC, storage, compatibility, and prototype scope

– `styling.md` – SCSS setup, shared variables path, and styling rules

– `repository-rules.md` – repository workflow rules for agents

– `release.md` – macOS release build and install flow

## Agent entry point

– `../AGENTS.md` – short agent entry point

## Reading order

1. `architecture.md`

2. `styling.md`

3. `repository-rules.md`

4. `release.md`

5. `../AGENTS.md`

## Apps

– `pet-overlay` – Angular renderer for the desktop pet overlay

– `pet-manager` – Angular renderer for pet management screens and dialogs

– `pet-host` – Electron host

– `pet-domain` – pure pet domain models, action rules, animation, atlas, and sprite logic
